# FlukeBench

A lightweight research tool for measuring response variability and identifying unusual generations in large language models.

## Paper

Check the project report:
[FlukeBench Paper (PDF)](https://drive.google.com/file/d/1VVgo8rkGYftlHLF4v9iKgGhMIpMABEY1/view?usp=sharing)

This is the first prototype developed during the Apart Research AI Safety Hackathon. Currently, it operates solely on semantic similarity and detects outliers using that metric. In the future, I plan to incorporate stance-based, stylistic, and topical similarity metrics to broaden the scope of the tool.
