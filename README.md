# FlukeBench

A lightweight research tool for measuring response variability and identifying unusual generations in large language models.

## Paper

Check the project report:
[FlukeBench Paper (PDF)](https://drive.google.com/file/d/1VVgo8rkGYftlHLF4v9iKgGhMIpMABEY1/view?usp=sharing)

This is the first prototype developed during the Apart Research AI Safety Hackathon. The current version focuses on semantic similarity analysis and in detecting outliers. Future versions may incorporate stance-based, stylistic and topical similarity metrics to enhance the scope of this tool.
